# ServUO Missing Features Checklist (OSI Updates)

This checklist compiles features, items, mobile creatures and systems introduced in **Ultima Online** (OSI) publishes that are **not present** in ServUO as of the latest ServUO Publish 57/58 (mid‑2020s). The list also includes community‑reported bugs and missing OSI mechanics. Developers can tick off each item as they implement or verify the feature in ServUO.
