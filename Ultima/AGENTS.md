# AGENTS Instructions for Ultima

- Library for reading Ultima Online client data used by other projects.
- Run `dotnet build Ultima/Ultima.csproj` after modifying code.

## Style

- Run `dotnet format Ultima/Ultima.csproj` before committing.
- Use 4 spaces for indentation, keep lines under 120 characters, and place braces on the same line.
- Use PascalCase for public types and methods.
- Use camelCase with a leading underscore for private fields.

## Testing

- No automated tests are available for this project.
- When tests are added, run `dotnet test Ultima` before committing.
